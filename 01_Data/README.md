# Dataset Information

## File
- `ecommerce_sales_data.csv`

## Description
This dataset represents simulated e-commerce sales transactions used for
analytics, KPI evaluation, and dashboard development.

## Contents
The dataset includes fields related to:
- Order details
- Sales revenue
- Profit
- Product categories
- Customer and regional attributes

## Source
The dataset was programmatically generated using Python for educational,
analytical, and professional portfolio purposes.

## Usage
This data is used to demonstrate:
- Data cleaning and preprocessing
- KPI calculation
- Sales and profitability analysis
- Dashboard and insight generation
- Data visualization
