# \## 🚀 Project Overview

# 

# \*\*Nexus Analytics\*\* is a comprehensive end-to-end data analytics project that demonstrates a real-world workflow, including:

# \* Data cleaning and modeling in Excel Power Pivot

# \* Creation of DAX measures for KPIs and YoY analysis
